Principe de fonctionnement :

Le programme IA regarde les meilleurs coups de l'humain (via la fonction Examiner_coup_humain). Il regarde à combien de coups le joueur humain  se trouve de la victoire.
Une fois ce nombre trouvé, l'IA calcule le nombre de coup avant la victoire de l'ordinateur (via la fonction examiner_coup_ordinateur) pour chacune des case disponibles (cad les cases n'ayant ni X ni O)

Enfin après ces 2 traitements, la fonction cherche la case ayant le nombre le plus bas (qui correspond au meilleur coup possible pour jouer en tant qu'ordinateur)

Fonction Intelligence_Artificielle($Tab = Tableau d'entier de 3*3)
  examiner_coup_humain($Tab)
  examiner_coup_ordinateur($Tab)
  $Valeur = 100 
  $i_trouve = 2, $j_trouve = 2
  for $i = 1 to 3
    for $j = 1 to 3
      if $tab[i][j] <> "X" and $tab[$i] [$j] <> "O" then
        if $tab[$i] [$j] < $valeur then
          $valeur = $tab[$i] [$j]
          $i_trouve = $i
          $j_trouve = $j
      endif
    next
  next
  // On modifie via l'interface la valeur du bouton situé en $i_trouve et $j_trouve avec la valeur 0 
FinFonction

Fonction Traitement_Ligne($ligne, $Colonne_Calcul, $Comparaison)
  // $Ligne = Ligne de traitement
  // $Colonne_Calcul = Référence colonne de la case qui doit être calculé
  // 
  // Exemple : Traitement_Ligne(1, 2) correspond à la première ligne et la cellule à calculer est la 2ème
  
  // Variables locales:
  // $rc = permet de renvoyer le nombre de coup avant que le joueur humain 
  // gagne (c'est à dire avant d'aligner 3 X sur la ligne <$ligne>)
  Select
    Cas $Colonne_Calcul = 1
      Si $tab[$ligne][2] == $Comparaison alors
        Si $tab[$ligne][3] == $Comparaison alors
          // Le joueur humain est à un coup de la victoire (-XX)
          $rc = 1
        Sinon
          // Le joueur humain est à 2 coups de la victoire (-X-)
          $rc = 2
        FinSi
      Sinon
        Si $tab[$ligne][3] == $Comparaison alors 
          // Le joueur humain est à 2 coups de la victoire (--X)
          $rc = 2
        Sinon
          // Le joueur humain est à 3 coups de la victoire (---)
          $rc = 3
        FinSi
      FinSi
    
    Cas $Colonne_Calcul = 2
      Si $tab[$ligne][1] == $Comparaison alors
        Si $tab[$ligne][3] == $Comparaison alors
          // Le joueur humain est à un coup de la victoire (X-X)
          $rc = 1
        Sinon
          // Le joueur humain est à 2 coups de la victoire (X--)
          $rc = 2
        FinSi
      Sinon
        Si $tab[$ligne][3] == $Comparaison alors 
          // Le joueur humain est à 2 coups de la victoire (--X)
          $rc = 2
        Sinon
          // Le joueur humain est à 3 coups de la victoire (---)
          $rc = 3
        FinSi
      FinSi

    Cas $Colonne_Calcul = 3
      Si $tab[$ligne][1] == $Comparaison alors
        Si $tab[$ligne][2] == $Comparaison alors
          // Le joueur humain est à un coup de la victoire (XX-)
          $rc = 1
        Sinon
          // Le joueur humain est à 2 coups de la victoire (X--)
          $rc = 2
        FinSi
      Sinon
        Si $tab[$ligne][2] == $Comparaison alors 
          // Le joueur humain est à 2 coups de la victoire (-X-)
          $rc = 2
        Sinon
          // Le joueur humain est à 3 coups de la victoire (---)
          $rc = 3
        FinSi
      FinSi
  FinSelect
  Retourner $rc
FinFonction
